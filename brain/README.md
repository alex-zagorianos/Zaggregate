# brain/ — open development journal

This folder, together with [`docs/handoffs/`](../docs/handoffs/), is an
intentional, unedited development journal. It holds the plans, design specs,
review reports, evaluation notes, and per-session handoffs that produced
Zaggregate — kept in the repo for transparency, not polish.

These are internal working notes: they are written session-by-session, may
reference in-progress ideas that were later changed or dropped, and are not
maintained as finished documentation. The polished, current docs live in
[`../README.md`](../README.md) and under [`../docs/`](../docs/) — start there if
you want to understand or use the app. Read here if you want to see how it was
actually built.

The canonical project status is [`project-status.md`](project-status.md); the
navigable index of everything is [`../_index.md`](../_index.md).
