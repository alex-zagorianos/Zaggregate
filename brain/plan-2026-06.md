## Phase 1 — Security & data integrity (TIER 1)

- ⬜ `browser_receiver.py` — URL-scheme allowlist in `_to_job_result` (reject non-http/https); `isinstance(j, dict)` guard; tighten CORS from `*` to a fixed allowed origin/loopback; explicit `host="127.0.0.1"`

- **Verify:** unit tests for scheme validation, cache-key uniqueness across salary_min, salary parsing (annual/hourly/K/garbage)

## Phase 2 — Rate limiting & resilience (TIER 1)

- ⬜ Shared `RateLimiter` (timestamp-at-start, loop-and-evict) — replace per-client deque logic

- ⬜ `requests.Session` + `urllib3 Retry` (429/5xx backoff) shared
- ⬜ `search_engine.py` — distinguish empty-results (`break`) from error (log+continue)
- ⬜ HTTPError guards `getattr(e.response,"status_code","?")`; `(item.get("company") or {})`; float coercion of salary fields; atomic cache writes
- **Verify:** unit tests for rate-limiter windowing, monthly counter rollover, salary float coercion
