## Direct Hermes head-to-head (test-02 tasks)

These three are what the local model is also solving. Documented to file:symbol precision.

### SEARCH-5 ‚Äî `search/usajobs_client.py::_normalize_location`

Removed the hardcoded `", OH"` append. The method now returns the caller's location verbatim (`return location.strip()`), so `Austin`‚Üí`Austin` and `Remote`‚Üí`Remote` instead of being silently rewritten to `Austin, OH` / `Remote, OH`.
See Claude's version: `git -C "E:\ClaudeWork\zag0005-allfixes" diff master -- search/usajobs_client.py`
Tests: `tests/test_usajobs_location.py` (4 ‚Äî comma-less unchanged, Remote not rewritten, comma kept, whitespace stripped).

### SCORE-7 ‚Äî `match/scorer.py::score_job` (size_adj)

Added the missing **101‚Äì250 band** (`size_adj = -2`), eliminating the dead zone between the `<=100 ‚Üí +4` and `>250 ‚Üí -6` bands. Bands are now monotonic **+8 / +4 / ‚àí2 / ‚àí6** (`<=30 / <=100 / <=250 / else`).
See Claude's version: `git -C "E:\ClaudeWork\zag0005-allfixes" diff master -- match/scorer.py`
Tests: `tests/test_scorer_compress.py::test_size_band_mid_large_gets_minus_two`, `::test_size_band_150_scores_below_50`, `::test_size_bands_monotonic`.

### SEARCH-6 ‚Äî `search/http_util.py::MonthlyQuota.decrement` + `search/jsearch_client.py`

New `MonthlyQuota.decrement(n=1)` refund method (lock-guarded, no-ops on a stale-month file). `JSearchClient.search` now wraps the `session.get` / `raise_for_status` / `.json()` in `try/except Exception ‚Üí self.quota.decrement(); raise`, so a failed request (network error or non-2xx after retries) refunds its reserved slot instead of permanently burning the 200/month free tier.
See Claude's version: `git -C "E:\ClaudeWork\zag0005-allfixes" diff master -- search/http_util.py search/jsearch_client.py`
Tests: `tests/test_quota_refund.py` (5 ‚Äî decrement restores, never below real count, stale-month no-op, jsearch refunds on failed request, no leak across failures).

### Foundation (supports SEARCH-8 / SCORE-3)

- `config.py` ‚Äî `DATA_DIR` (read-only `_MEIPASS`/repo root) vs `WRITABLE_DIR` (`<exe>/JobProgram`, falling back to `%LOCALAPPDATA%`) split; `BASE_DIR` kept as a back-compat alias; import-time `mkdir` replaced by best-effort `ensure_writable_dirs()`.
