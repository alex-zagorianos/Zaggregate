## Pipeline

Wide-net search ‚Üí preferences JSON hard-gate ‚Üí local 0‚Äì100 match scoring (`match/scorer.py`) ‚Üí deduped **Inbox** ‚Üí AI fine-rank to `preferences.md` (clipboard bridge default, optional API auto) ‚Üí **Apply Queue** GUI tab with resume prompts + "Mark Applied ‚Üí Next". Free no-key sources: The Muse, RemoteOK.

**Two distribution channels on one engine + data folder** (2026-06-22): (1) the **EXE** with hybrid AI for non-technical friends ‚Äî `py build_package.py` ‚Üí `dist/JobScout.zip`; (2) the **MCP server + `find-jobs` Claude Code skill** (`mcp_server.py` + `claude-code/`) where Claude Code itself is the ranker. Friends edit `data/preferences.md` (NL profile) + `data/preferences.json` (hard filters); their own data never ships.

**Coverage engine + AI re-rank (2026-06-22, Session 13):** a **3-leg coverage benchmark** (`coverage/`) rates how completely a search finds an area's jobs (reference-proxy ‚à™ capture-recapture ‚à™ JOLTS gate), keyed by a stable `job_key`. A generic **discovery funnel** (`discover/`) + Tier-1 ATS scrapers (`scrape/`) + free aggregators (`search/`) + geo/freshness raise that score ‚Äî **every source gated by a lift-test** proving it doesn't lower coverage. An **AI re-rank round-trip** (`rerank/`, `ranker.py`) exports the inbox + a versioned prompt for any AI, imports the returned scores (validated, `job_key`-joined), snapshots to `score_history`, and supports undo.
